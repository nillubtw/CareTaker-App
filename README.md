# CareTaker-App
# Assistive Wearable Alert System

An **IoT-based assistive safety system** designed for disabled and elderly individuals that enables **hands-free emergency communication** with caretakers using a wearable device and a real-time mobile dashboard.

---

## Project Overview

This project integrates an **ESP32-based wearable**, a **Node.js gateway**, **Firebase Realtime Database**, and an **Android caretaker application** to create a **real-time, reliable emergency alert system**.

Users can trigger alerts using **gestures or sound patterns**, and caretakers receive instant notifications with the ability to **acknowledge alerts**, which are then reflected back on the wearable device.

## Alert Triggers

Alerts are generated by the wearable device when it detects:

- Sound patterns (Short hum / Long hum)
- Hand gestures (Left / Right)
- Emergency gestures (customizable)

Each alert contains:
- Alert type
- Timestamp
- Device ID
- Acknowledgment status

## Acknowledgment Flow

1. Caretaker receives alert on Android app  
2. Caretaker taps **ACKNOWLEDGE**  
3. Firebase updates alert status
